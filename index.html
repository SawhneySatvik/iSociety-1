iSociety

iSociety is an Object Oriented Design and Programming Project on a housing society management system. It is a systemised way of putting everything together from managing your apartment/houses to payment of various bills, etc.





Code

iSociety is one of a kind project made using C++ programming language. Take a look into the code, which made it possible to run iSociety successfully and smoothly.
